# SecureLocker
由于经常需要和朋友交换一些比较秘密的信息，比如密码、重要文件和数字货币的密钥等。所以专门写了个小工具，用来和其他人交换加密的数据。
该软件采用了非对称RSA算法，自动生成密钥对（公钥和私钥），只要把公钥发给对方，对方用公钥加密，就只有你自己能进行解密。
先发布一个版本，看看大家反馈，后面再逐步改进。

## 名称简介
SecureLocker传达了保护和安全的含义。这个名字结合了"secure"（安全）和"locker"（储物柜）两个词语，引起了关于保护个人物品的关注，同时暗示着文件锁的功能。
SecureLocker的含义是一个安全的储物柜或保险箱，用于储存和保护重要的文件和数据。它可以用于加密和锁定文件以防止未经授权的访问，并提供额外的安全层，确保敏感数据不被泄露或遭到损坏。
这个名字还可以使人们联想到实体储物柜中物品的保护，与数字化世界中的数据保护相对应。它传递了对隐私和安全的关注，适用于个人、企业和组织在日常工作中保护重要文件时使用。

## 使用
该软件无需安装，可以直接使用。绿色运行。
